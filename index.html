🚀 域名部署指南

1. 在腾讯云/阿里云购买 .tech 域名 aidrawing.user.tech 和 aidrawing.designer.tech

2. 开通腾讯云 COS 静态网站托管，将 website/user/ 和 website/designer/ 分别上传

3. 在域名解析中添加 CNAME 记录，分别指向两个 COS 桶的访问域名

4. 申请并绑定 SSL 证书（腾讯云免费 TrustAsia 证书）以启用 HTTPS

5. 等待 DNS 解析生效（通常 5–30 分钟）

💡 如需后端反馈存储，建议接入腾讯云函数 SCF + TencentDB 数据库替代 localStorage
